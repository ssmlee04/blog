# {{ page.title }}

#### The following commands are useful sometimes

```
⌘ + option + ← (or →)
```

Switch between tabs in the current window.

```
⌘ + ← (or →)
```

Go to the previous screen or the next screen.

```
⌘ + 1, 2, 3, 4, ...etc
```

Choose the first, second, third tabs... etc
